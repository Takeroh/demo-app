# Re:Trip
**あの旅行を、もう一度。**

Re:Tripは大切な旅行の思い出を振り返るサービスです。

あなたも旅路をたどってあのときの気持ちを思い出しませんか？

## 概要
写真をアップロードするだけで、音楽やスタンプとともにスライドショーで旅行の思い出を振り返ることができます。地図上で旅行の足跡を辿ることもできます。

## 機能

### 共通
| 機能 | 説明|
| --- | :--- |
| 画像加工 | AIによって判定された最適な画像加工を適用します。 |
| 撮影日時・撮影場所表示 | 写真が撮影された日時および撮影場所を確認できます。 |

### スライドショー
| 機能 | 説明|
| --- | :--- |
| 音楽 | AIが写真の風景を判定し、風景にあわせた音楽を再生します。 |
| スタンプ | AIが写真から感情を推定し、感情にあわせたスタンプを表示します。 |

### 地図
| 機能 | :説明 |
| --- | --- |
| 旅行経路表示 | 写真の撮影日時情報および位置情報をもとに、地図上に旅行の経路を表示します。 |
| 写真表示 | 地図のピンをクリックするとその場所で撮影した写真を確認することができます。 |


## 環境構築
### 1. npmのインストール
ターミナルで以下のコマンドを実行してnpmをインストールします。
```
npm install
```
### 2. .envファイルの作成
ルートディレクトリに.envファイルを作成し、以下の内容を記載してください。
```
OPENAI_API_KEY = "your_OpenAI_API_key"
GOOGLE_MAPS_API_KEY = "your_Google_Maps_API_key"
```

### 3. 開発サーバーの立ち上げ
以下のコマンドを実行して開発サーバーを立ち上げます。
```
npm run dev
```

### 4. ブラウザ表示
右下に表示されたボタンまたはターミナル上のリンクからブラウザで表示します。

## 操作方法
### 1. 画像のアップロード
「写真のアップロード」ボタンからアップロードする写真ファイルを選択してください。

ファイルを選択したら「選択した写真をアップロード」ボタンを押してください。

### 2. 画像処理
サーバーで画像処理が完了したら自動的に画面が遷移します。

### 3. スライドショー
旅行の思い出を写真のスライドショーとして振り返ることができます。
矢印ボタンを押すか、左右にスワイプまたはスクロールすることで前後のスライドに移動できます。
写真の下のページネーションボタンからもスライドの移動が可能です。

### 4. 地図
「旅行経路を表示」ボタンから、旅行の経路を地図上で振り返ることができます。
地図上のピンをタップまたはクリックでその地点で撮影された画像を確認することができます。
「スライドショーを表示」画面からスライドショー画面に戻ることができます。

### 5. データの削除
ページ下部の「リセット」ボタンを押すとすべてのデータが削除され、トップページに遷移します。
